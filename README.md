## Project setup
```
npm install
```
Then create a Firebase project and paste configuration in main.js

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
